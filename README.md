# Noticias IRO

Una aplicación de Android para ver las últimas noticias utilizando la API de NewsAPI.

## Características
- Muestra las últimas noticias de diferentes fuentes
- Interfaz moderna y fácil de usar
- Soporte para arquitecturas de 32 y 64 bits
- Actualización con gesto de deslizar hacia abajo
- Carga de imágenes con Glide

## Requisitos
- Android 5.0 (API 21) o superior
- Conexión a Internet

## Configuración
1. Obtén una clave de API de [NewsAPI](https://newsapi.org/)
2. Reemplaza `YOUR_API_KEY` en `NewsApiService.kt` con tu clave de API

## Compilación
Este proyecto utiliza GitHub Actions para compilar automáticamente el APK. Puedes descargar el APK desde la sección de Actions en GitHub.
